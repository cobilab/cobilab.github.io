Fast Sequence Characterization Toolkit

About AlcoR

AlcoR is a fast and effective solution for characterizing sequences, particularly in situations where new or unknown sequences are present. It is highly sensitive, quick, and reliable, making it an ideal choice for use in Telomere-to-telomere (T2T) sequencing and assembly.


AlcoR is implemented in C language using multi-threading to increase its speed, and it is flexible enough to be used for a variety of purposes without any external dependencies. It can process any sequence in FASTA format, making it a convenient choice for a wide range of applications.

Features

AlcoR is a powerful and efficient solution for characterizing and analyzing nucleotide sequences. It is implemented in C language and uses multi-threading to increase its computational speed and power. It can process any sequence in FASTA format and offers a range of advanced features for sequence analysis, including extraction using positional coordinates, low-complexity mapping and bidirectional complexity profiles, simulation with various methods, and visualization with an SVG file map. Whether you are working with known or unknown sequences, AlcoR is a reliable and practical choice for researchers and scientists working with DNA and genomics data.

Modules

The AlcoR tool is a powerful solution for analyzing nucleotide sequences. It has four sub-menus that provide advanced features for analyzing and manipulating sequences.

  • Extract
    Extracts a sequence from a FASTA file using positional coordinates, independent of the existing headers.
  • Mapper
    Computes low-complexity regions in a FASTA read, does sequence masking and provides bidirectional complexity profiles and structural similarity analysis.
  • Simulation
    Simulates FASTA sequences using a variety of methods, including file extraction, random generation, and sequence modeling. It also allows users to apply specific SNPs probability mutations.
  • Visual
    Computes an SVG file map containing the low-complexity regions, providing a useful visualization of the analysis results.

Installation

GTO2 can be installed using the Cobilab channel from Conda:

conda install -y -c bioconda alcor

Or through the GitHub repository:

sudo apt-get install cmake git
git clone https://github.com/cobilab/alcor.git
cd alcor/src/
cmake .
make
